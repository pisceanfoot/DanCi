项目记录
双击选中单词，页面高亮并记录保存选中的单词保存到生词本中。

支持：

1. 高亮生词
2. 在线及时朗读
3. 单词快速在线翻译

使用方法：

1. 双击取词
2. 双击取消
3. Chrome 浏览器右上角在线翻译

OPEN SOURCE: https://github.com/pisceanfoot/DanCi
